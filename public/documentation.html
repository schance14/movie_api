FilmFinder App

This is a web application that stores data about different movies as well as provides users with a variety of information on a list of movies.

App Documentation

Request Url HTTP Method Request Data Format Response Data Bodt Format
Get list of all movies /movies GET none json data holding data about all movies.
Get data about a single movie by the title /movies/:title GET none a json object holding data about a single movie title.
Get data about a genre by genre name /movies/genre/:genreName GET none a json object holding data about a genre.
Get data about a director by director name /movies/directors/:directorName GET none a json object holding data about a director.
Create a new user to register /users POST json object holding user data information json object holding user data with an ID and array for list of favorite movies.
Update user information /users/:id PUT json objet holding all the user information with the information that needs to be changed. json object holding all user information along with updated information.
Adding a favorite movie to the user's favorite list /users/:id/:movieTitle POST none json object holding user information with their favorite movie list updated.
Removing a movie from the user's favorite list /users/:id/:movieTitle DELETE none json object with all user information including their updated favorites list.
Exisiting user to deregister account /users/:id DELETE none a message indicating that the user was succesfully deregistered.
